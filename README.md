# Proyecto Final de Git

El algoritmo de Shor es un algoritmo cuántico desarrollado por el matemático Peter Shor en 1994, diseñado para encontrar los factores primos de un número entero grande en tiempo polinomial. Su capacidad para realizar esta tarea de manera exponencialmente más rápida que los ordenadores clásicos amenaza la seguridad de los sistemas criptográficos modernos
